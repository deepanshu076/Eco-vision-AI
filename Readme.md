We are creating **EcoVision AI 🌍**, a smart waste segregation ♻️ and carbon footprint analyzer 🌱 that empowers individuals to take measurable climate action.
By combining deep learning 🤖, image recognition 📸, and sustainability analytics 📊, our platform turns everyday waste disposal into meaningful environmental impact 🌎✨.

**EcoVision AI** is an AI-powered full-stack web application 💻 that uses computer vision 👁️ to:

* Classify waste into:

  * Biodegradable 🌿
  * Recyclable ♻️
  * Hazardous ☣️
* Estimate carbon footprint reduction 🌍⬇️
* Support multi-user authentication 🔐
* Prevent duplicate image uploads using perceptual hashing 🛑📷
* Provide a sustainability analytics dashboard 📊🌱

The system leverages Transfer Learning (MobileNetV2 / EfficientNet) 🧠 for image classification and delivers a gamified eco-impact experience 🏆🌿, encouraging users to build sustainable habits while tracking their positive environmental contribution.
